Asistente de atención al cliente para un emprendimiento de cartelería de neón personalizado, capaz de:

- responder consultas de clientes
- asesorar en diseños, tamaños y precios
- explicar tiempos de producción y envío
- escalar a humano cuando haga falta
- mantener tono de marca

Preguntas base para construir el bot:

- ¿Qué vende el negocio?
- ¿Qué preguntan siempre los clientes?
